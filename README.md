# grid_road

> An aggregation of the [Global Roads Inventory Project (GRIP)](https://www.globio.info/download-grip-dataset) dataset, to the [PRIO GRID](https://www.globio.info/download-grip-dataset)  dataset.

## Data
- road_density.geojson: GRIP's global raster datasets of road density at a 5 arcminutes resolution (~8x8km), aggregated to the PRIO GRID resolution (~55x55km) by gid.
